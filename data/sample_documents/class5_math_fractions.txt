# Class 5 Math - Chapter: Fractions

## Learning Objectives
- Understand the concept of fractions
- Learn to add and subtract fractions
- Compare fractions with different denominators
- Solve word problems involving fractions

## Key Concepts
1. A fraction represents a part of a whole
2. The numerator represents the number of parts we have
3. The denominator represents the total number of equal parts
4. Equivalent fractions
5. Simplifying fractions to lowest terms

## Activities
1. Using fraction tiles to visualize fractions
2. Creating fraction models with paper folding
3. Playing fraction games with cards
4. Drawing fraction pictures

## Assessment
1. Worksheet on identifying fractions
2. Quiz on comparing fractions
3. Word problems using fractions in daily life
4. Group activity on creating fraction posters
